Planverkehr

Hauptprojekt des Praktikums ist ein Verkehrssimulationsspiel. Es orientiert sich lose an den Computerspiele n”Transport Tycoon™“  von 1994 und ”Simutrans“  von 1997. In  diesem  Spiel  verwaltet  man  das  Verkehrsnetz  (Bus,  Bahn  und  Flugverkehr)  in  einer Region mit mehreren Industrieanlagen. Die Simulation läuft fortwährend ab, die Spielkarte ist kästchenbasiert.
